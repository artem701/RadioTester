Set commands: for manipulating transmitter's settingsset channel [auto | <channel> | <start> <end>]  sets radio channel for both transmitter and receiver  can be setted automatically and automatically in the given range   (according to noise level)set delay [delay ms]  sets delay between two packetsset power <power option>  sets transmission powerset length [length option]  sets length of a radio packetset pattern <pattern option>  sets the transmission pattern for generating packets  info  displays current transmisson settings  Test commands: for running radio tests  test  launches a series of tests on the current channel and power,  reporting statisticstest channel  tests current channel's characteristicstest (all | <channel> | <begin> <end>)  tests given channels (all | specific | range), chooses the best one
